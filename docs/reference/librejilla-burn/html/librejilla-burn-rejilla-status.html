rejilla_status_get_result ()

RejillaBurnResult   rejilla_status_get_result           (RejillaStatus *status);

After an object (see rejilla_burn_track_get_status()) has been requested its status, this function returns that status.

status :

 a RejillaStatus.

Returns :

 a RejillaBurnResult. REJILLA_BURN_OK if the object is ready. REJILLA_BURN_NOT_READY if some time should be given to the object before it is ready. REJILLA_BURN_ERR if there is an error.
