rejilla_track_data_cfg_span ()

RejillaBurnResult   rejilla_track_data_cfg_span         (RejillaTrackDataCfg *track,
                                                         goffset sectors,
                                                         RejillaTrackData *new_track);

Creates a new RejillaTrackData (stored in new_track) from the files contained in track. The sum of their sizes does not exceed sectors. This allows to burn a tree on multiple discs. This function can be called repeatedly; in this case if some files were left out after the previous calls, the newly created RejillaTrackData is created with all or part of the remaining files.

track :

 a RejillaTrackDataCfg

sectors :

 a goffset

new_track :

 a RejillaTrackData

Returns :

 a RejillaBurnResult. REJILLA_BURN_OK if there is not anymore data. REJILLA_BURN_RETRY if the operation was successful and a new RejillaTrackDataCfg was created. REJILLA_BURN_ERR otherwise.
